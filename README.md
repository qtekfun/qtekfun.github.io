# qtekfun.github.io

Personal blog & technical notes — built with Beautiful Jekyll, hosted on GitHub Pages.

**Live site → [qtekfun.com](https://qtekfun.com)**

## Stack

| | |
|---|---|
| Generator | Jekyll + Beautiful Jekyll 6.0.1 |
| Hosting | GitHub Pages |
| Topics | DevOps · Self-hosted infrastructure · Linux · Privacy |

## Local development

```bash
bundle install
bundle exec jekyll serve
# open http://localhost:4000
```
