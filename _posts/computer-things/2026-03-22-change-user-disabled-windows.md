title: "How to change to another user in Windows 11 if `Switch User` is disabled?"
categories: computer-things
---

Sometimes you need to login with your admin user and you don't want to sign out with your actual user to not close al
your work in progress. But, some company policy disabled the option to switch user from the menu. There is still one
secret option to use it.

Open `win + R` dialog and then write `tsdiscon`

That will do the trick for you and get you to the "Other User" login screen.
